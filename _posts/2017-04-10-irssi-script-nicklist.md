![irssi](/image/irssi-script-nicklist-01.png)
1. Go to configuration directory
```bash
$ mkdir ~/.irssi/scripts
$ cd ~/.irssi/scripts
```
2. Download the script
```bash
$ wget http://scripts.irssi.org/scripts/nicklist.pl    
```
3. Run irssi
```bash
$ screen irssi
$ /run nicklist
$ /nicklist screen
```
